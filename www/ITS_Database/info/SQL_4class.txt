    <tr class="header-bg">
      <td class="w-20">指令 (Command)</td>
      <td class="w-40">公式 / 語法結構</td>
      <td class="w-40">參數與功能說明</td>
    </tr>
  </thead>
  <tbody>
    <tr class="category-title">
      <td colspan="3">一、 DDL (Data Definition Language) - 資料定義語言：定義資料庫結構</td>
    </tr>
    <tr>
      <td><b>CREATE</b></td>
      <td class="code-font">CREATE TABLE [表名] ([欄位] [型別] [約束])</td>
      <td>建立新的資料表、視圖或預存程序。</td>
    </tr>
    <tr>
      <td><b>ALTER</b></td>
      <td class="code-font">ALTER TABLE [表名] ADD/DROP COLUMN [欄位]</td>
      <td>修改現有的資料庫結構，例如增加或刪除欄位。</td>
    </tr>
    <tr>
      <td><b>DROP</b></td>
      <td class="code-font">DROP TABLE/PROCEDURE [名稱]</td>
      <td>完全刪除物件（結構與內容皆不保留）。</td>
    </tr>
    <tr>
      <td><b>TRUNCATE</b></td>
      <td class="code-font">TRUNCATE TABLE [表名]</td>
      <td>快速清空整張表的記錄，但保留表格結構。</td>
    </tr>

    <tr class="category-title">
      <td colspan="3">二、 DML (Data Manipulation Language) - 資料操作語言：處理記錄內容</td>
    </tr>
    <tr>
      <td><b>INSERT</b></td>
      <td class="code-font">INSERT INTO [表名] ([欄位]) VALUES ([值])</td>
      <td>將新記錄插入資料表。</td>
    </tr>
    <tr>
      <td><b>UPDATE</b></td>
      <td class="code-font">UPDATE [表名] SET [欄位]=[值] WHERE [條件]</td>
      <td>修改現有資料；省略 WHERE 會更新全表。</td>
    </tr>
    <tr>
      <td><b>DELETE</b></td>
      <td class="code-font">DELETE FROM [表名] WHERE [條件]</td>
      <td>刪除表中的特定記錄。</td>
    </tr>

    <tr class="category-title">
      <td colspan="3">三、 DQL (Data Query Language) - 資料查詢語言：檢索資料</td>
    </tr>
    <tr>
      <td><b>SELECT</b></td>
      <td class="code-font">SELECT [欄位] FROM [表名] WHERE [條件]</td>
      <td>從資料庫中檢索資料，是最常用的指令。</td>
    </tr>
    <tr>
      <td><b>JOIN</b></td>
      <td class="code-font">SELECT * FROM A JOIN B ON A.ID = B.ID</td>
      <td>結合多張表的資料（如 INNER, LEFT JOIN）。</td>
    </tr>

    <tr class="category-title">
      <td colspan="3">四、 DCL (Data Control Language) - 資料控制語言：權限管理</td>
    </tr>
    <tr>
      <td><b>GRANT</b></td>
      <td class="code-font">GRANT [權限] ON [物件] TO [使用者]</td>
      <td>賦予使用者存取特定資料庫物件的權限。</td>
    </tr>
    <tr>
      <td><b>REVOKE</b></td>
      <td class="code-font">REVOKE [權限] ON [物件] FROM [使用者]</td>
      <td>撤銷先前賦予使用者的權限。</td>
    </tr>

    <tr class="category-title">
      <td colspan="3">五、 TCL (Transaction Control Language) - 交易控制語言：確保原子性</td>
    </tr>
    <tr>
      <td><b>BEGIN TRANSACTION</b></td>
      <td class="code-font">BEGIN TRANSACTION</td>
      <td>標記交易處理的開始點。</td>
    </tr>
    <tr>
      <td><b>COMMIT</b></td>
      <td class="code-font">COMMIT</td>
      <td>永久儲存交易期間的所有變更。</td>
    </tr>
    <tr>
      <td><b>ROLLBACK</b></td>
      <td class="code-font">ROLLBACK</td>
      <td>將變更復原至交易前的狀態（撤銷操作）。</td>
    </tr>
  </tbody>
</table>
