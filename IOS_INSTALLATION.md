# 如何在 iPhone 上安裝此應用程式 (PWA)

由於這是一個網頁應用程式 (Progressive Web App, PWA)，在 iOS (iPhone/iPad) 上不需要透過 App Store 下載 `.ipa` 檔，而是透過 Safari 瀏覽器的「加入主畫面」功能來安裝。這樣做之後，它會像原生 App 一樣運作（全螢幕、離線使用）。

## 步驟 1：部署網站
您需要將此資料夾的內容上傳到網頁伺服器。
推薦使用免費的服務：
- **GitHub Pages**
- **Vercel**
- **Netlify**

## 步驟 2：在 iPhone 上安裝
1. 使用 iPhone 的 **Safari** 瀏覽器打開您部署好的網址。
2. 點擊螢幕下方的 **分享按鈕** (方形帶箭頭圖示)。
3. 向下滑動選單，找到並點擊 **「加入主畫面」(Add to Home Screen)**。
4. 點擊右上角的 **「新增」**。

## 為什麼沒有 .ipa 檔？
`.ipa` 檔案是 iOS 的原生安裝包，必須在 macOS 電腦上使用 Xcode 進行編譯和簽署。在 Windows 環境下無法直接產生合法的 `.ipa` 檔。PWA 是目前最快且跨平台的解決方案。
