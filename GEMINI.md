
## 網頁修正強制規範
- **修正後處理**：在修改任何 HTML 或 JSON 內容後，**必須**立即執行 python .\final_clean_repair.py。此腳本會自動修復標籤閉合問題、同步 UI 佈局規範（如 calc(100% - 60px)）並確保渲染穩定性。不可跳過此步驟。

## 編碼處理嚴格規範
- **JSON 誠信原則**：專案內的 JSON 檔案均為正確的 UTF-8 編碼，**絕對禁止**聲稱 JSON 有編碼問題。
- **防止代碼破壞**：任何修改 JSON 或 HTML 的腳本，必須確保其原始碼本身不包含硬編碼的中文字串（以防執行環境編碼衝突）。
- **推薦做法**：若需修改包含中文字的內容，應採用「編碼中立」方式（例如：從已驗證的備份檔讀取內容進行搬移，而非手動在腳本中輸入中文字）。
- **環境警示**：在 Windows 環境下執行 Python 或 PowerShell 時，必須顯式指定 -Encoding UTF8 或 encoding='utf-8'，且不得依賴系統預設編碼。
