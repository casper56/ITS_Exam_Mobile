# ITS 認證模擬測驗系統 (V3.2 更新)

## 🏆 專業介面與列印規範
本系統 V3.2 版本已統一全科目介面規格，維護時請參閱 `spec.md`：
- **按鈕標準**：寬 25px / 高 65px / 圓角 15px，禁止 Hover 變形。
- **佈局標準**：統一採用 30px 側邊留白與水平置中邏輯。
- **字體標準**：題目 #000 純黑，1.05rem；代碼塊遵循「去粉紅且高亮清晰」原則。

## 🏆 專業列印格調規範 (1:1 復刻 PDF 格調.PNG)
系統產出的 PDF 講義必須嚴格遵守以下視覺規範，以維持專業學術感：

- **背景色**：純白背景，禁止任何深色標題條或粉紅色塊。
- **題目標號**：採用 `52. 【CH05】題目內容` 格式，標號加粗。
- **選項樣式**：左側縮排 20px，文字清晰。
- **正確答案條**：
  - **綠色文字** (`#198754`)。
  - **左側裝飾線**：5px 粗綠色實線 (`border-left`)。
  - **背景**：純白 (不可有底色)。
- **解析區塊**：
  - **淺灰色背景** (`#f8f9fa`)。
  - **圓角框**：10px 圓角 (`border-radius`)。
  - **邊框**：1px 極淺灰邊框 (`#eeeeee`)。
  - **代碼塊**：繼承解析盒顏色，徹底禁止 Bootstrap 預設的粉紅色文字。

## 🎨 如何自訂題目文字顏色
如果您需要強調題目中的特定文字，可以直接在 JSON 檔案的 `question` 欄位中使用 HTML 的 `<span>` 標籤與 `style` 屬性。支援 `blue`, `#330909`, `rgb(51, 9, 9)` 等格式。

## 📊 科目出題配置與要求 (V3.3 智能抽題準則)

### 1. 官方與補充題比例
模擬考試全面採用 **90% 官方流出題 / 10% 補充題** 的比例：
- **40 題模式**：36 官方 / 4 補充。
- **50 題模式**：45 官方 / 5 補充。

### 2. 分類平衡與權重限制
為了確保知識點覆蓋全面且不偏食，抽題演算法遵循以下百分比限制：

| 限制條件 | ITS Python | 其他科目 | 說明 |
| :--- | :---: | :---: | :--- |
| **分類下限 (Min)** | **5%** | **5%** | 確保小章節一定會出題。 |
| **分類上限 (Max)** | **30%** | **25%** | 防止題目多的分類佔滿整卷。 |

*註：若某類別官方題庫不足 5%，則採取「全抽」策略以滿足多樣性。*

### 3. 各科官方題範圍界定
| 科目名稱 | 官方題 ID 範圍 |
| :--- | :--- |
| **ITS 系列 (AI除外)** | **ID 1 - 69** |
| **ITS AI (Artificial Intelligence)** | **ID 1 - 118** |
| **微軟 900 系列 / 其他** | **ID 1 - 100** |
