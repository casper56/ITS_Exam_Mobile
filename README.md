# ITS 認證模擬測驗系統 (V3.4 全屏外擴版)

## 🏆 專業介面與列印規範 (V3.4 更新)
本系統 V3.4 版本已優化全科目介面，維護時請嚴格遵守：
- **全屏外擴 (Edge-to-Edge)**：精確移除所有 `max-width: 1200px` 限制。題目區塊隨網頁寬度自由延伸。
- **邊距對齊 (30px Rule)**：容器統一採用 `width: calc(100% - 60px); margin: auto;`。
- **按鈕同步**：側邊導覽按鈕固定於 `left: 280px`（避開側邊欄），與內容區保持 **5px** 固定間隙並同步位移。
- **字體標準**：題目 #000 純黑，1.05rem；代碼塊遵循「去粉紅且高亮清晰」原則。

## 🏆 專業列印格調與預覽禁忌
系統產出的 PDF 講義必須嚴格維持專業學術感：
- **禁止結構性替換**：嚴禁替換現有 `#review-area` 邏輯。
- **去粉紅規則**：代碼塊禁止 Bootstrap 預設的粉紅色文字，背景必須為純白或淺灰 (#f8f9fa)。
- **正確答案條**：必須為「純白背景、綠色文字 (#198754)、左側 5px 綠實線」，禁止私自加入底色。
