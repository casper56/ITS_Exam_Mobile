# ITS 認證模擬測驗系統 (V3.4.6 核心引擎版)

## 🏆 專業介面與佈局鎖定規範
本系統 V3.4.6 版本已實現物理級佈局鎖定，維護時請嚴格遵守：
- **物理鎖定 (Layout Shift Prevention)**：採用 `scrollbar-gutter: stable` 確保垂直捲軸出現時內容不跳動。
- **全屏外擴 (Edge-to-Edge)**：移除所有 `max-width` 限制。容器統一採用 `width: calc(100% - 50px); margin: auto;`。
- **邊距對齊 (25px Rule)**：確保左右精確留白 **25px**。
- **導覽按鈕配置 (Interactive Scheme)**：
    - **平時 (Mouse Out)**：寬度 **25px**，背景為 **灰色 (`rgba(108, 117, 125, 0.7)`)**。按鈕邊緣與內容區完美貼合。
    - **觸碰 (Mouse In)**：寬度 **30px**，背景變更為 **藍色 (`#0d6efd`)**。具備 `width 0.2s` 平滑縮放感。
- **字體與顯示**：題目 #000 純黑，1.05rem；代碼塊強制換行 (`pre-wrap`) 並遵循「去粉紅」原則。

## 🏆 專業列印格調與預覽禁忌
系統產出的 PDF 講義必須嚴格維持專業學術感：
- **正確答案條**：必須為「純白背景、綠色文字 (#198754)、左側 5px 綠實線」，禁止私自優化或加入底色。
- **答案忠實顯示**：列印報告中的「正確答案」文字會根據 JSON 值自動映射 (Alpha/Num)。

## 📊 數據規範 (V3.4.6)
### 1. JSON 格式要求 (核心標準)
所有題目必須存儲於各考科目錄下的 `questions_*.json` 中：
- **題目內容 (`question`)**：建議採用 **字串陣列 (Array of Strings)** 格式，每行一個元素，系統將自動換行渲染。
- **編碼**：必須為 **UTF-8 (無 BOM)**。
- **選項編號標註 (新版預設)**：
    - **不設定 (預設)**：自動顯示為 **`(A)`, `(B)`, `(C)`** 格式。
    - **`"labelType": "num"`**：切換為數字編號 **`1.`, `2.`, `3.`**。
- **答案自動對齊 (UI 智慧轉換)**：系統會根據編號模式自動轉換顯示文字 (A->1 或 1->A)。

### 2. 官方題比例與 CUTOFF (V3.4.6)
- 系統自動維持 **95% 官方流出題 / 5% 補充題** 比例 (抽題邏輯優化)。
- **各科 CUTOFF 設定 (最新版)**：
  - **ITS Python**：ID **1 - 91** 為官方題。
  - **ITS AI**：ID **1 - 118** 為官方題。
  - **ITS Database**：ID **1 - 69** 為官方題。
  - **ITS Software Dev**：ID **1 - 101** 為官方題。
  - **微軟 900 系列 / 其他**：ID **1 - 100** 為官方題。
